UMS Bot Core ("the Bot") is designed to operate with minimal data usage and does not collect or store personal information beyond what is required for core functionality.

## 1. Information the Bot Collects
The Bot only stores the following non-sensitive information:

- Discord User ID (for linking players to tournaments)
- Username/nickname at the moment of onboarding
- Region and optional claimed rank (as entered by the user)
- Tournament match data such as player assignments and results
- Server configuration settings created by admins

No other personal data is collected.

## 2. Information the Bot **Does Not** Collect
The Bot does **not** collect, store, or process:

- Email addresses
- Phone numbers
- Payment information
- IP addresses
- Message content
- Direct messages
- Voice data

The Bot does not use cookies, trackers, analytics scripts, or share any data with third parties.

## 3. How Data Is Stored
All data is stored locally on the server hosting the Bot using SQLite.
No data is transmitted to external servers or external services.

## 4. How Long Data Is Retained
Data is retained only as long as required for tournament functionality.
Admins may delete bot data at any time using the Bot's built-in factory reset features.

## 5. Data Removal
Server owners may remove all saved data by:

- Kicking the bot
- Running the built-in factory reset command
- Deleting the bot's local storage directory if self-hosted

Users may request deletion of their data through the server owner hosting the Bot.

## 6. Contact
For questions or concerns, please open an issue on the GitHub repository:
https://github.com/Comradecast/UMS
